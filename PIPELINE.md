# Reproduction Pipeline

This file documents the exact sequence of steps required to reproduce all data,
inference results, and figures in this project from scratch.

**Authors:** Nozhan Balafkan and Dr. Johannes Buchner

---

## Stage 0 — Environment

Required packages: `numpy`, `scipy`, `matplotlib`, `cmdstanpy`, `pandas`, `arviz`

Install:
```bash
pip install numpy scipy matplotlib cmdstanpy pandas arviz
```

---

## Stage 1 — HMC Inference (generates large NPZ files)

These are the computationally expensive runs. Each produces a large NPZ file
containing posterior samples for all 6000 draws (T=100 time steps each).

| Script | Parameters | Output | Est. runtime |
|--------|-----------|--------|-------------|
| `code_backup/count_inference_hmc.py` | NH=21, TAU=2, PHI=-3 | `*_Nh21_Tau2_Phi-3*.npz` | ~hours |
| `code_backup/count_inference_hmc.py` | NH=22, TAU=38, PHI=-4 | `*_Nh22_Tau38_Phi-4*.npz` | ~hours |

The exact paths to the large NPZ files are documented in the `README.md` of
each reproducibility directory under `plots/`.

---

## Stage 2 — CSV Summary Files

These are small summary files (means, credible intervals) extracted from the
HMC posterior. They are used as the authoritative source for table values in
the paper and for CI markers in the tau posterior plots.

| File | Parameters |
|------|-----------|
| `code_backup/plot/summary_Nh_22_and_Tau_38_and_phi_-4.csv` | NH=22, TAU=38, PHI=-4 |

---

## Stage 3 — PSD Cache Files (fast, ~minutes)

These cache files store pre-computed periodogram statistics across 1000
posterior draws. They are generated automatically on the first run of any
plot script that needs them.

| Cache file | Generated by |
|-----------|-------------|
| `results/cache_true_fits_Nh21_Tau2_Phi-3.npz` | `nh_phi_psd_beta_recovery_methodB_paper.py` (first run) |
| `results/cache_sample_psds_Nh21_Tau2_Phi-3_n1000_seed42.npz` | Same script (first run) |

---

## Stage 4 — Paper Figures (in order)

Run each script from its reproducibility directory. Each directory is
self-contained with script + data + outputs.

### Figure 1 — τ Posterior (NH=22, TAU=38, PHI=-4)
```bash
cd plots/tau_posterior_tau38/
python tau_posterior_paper.py
```
**Requires:** Stage 1 NPZ (Nh22 Tau38), Stage 2 CSV
**Outputs:** `tau_posterior_Nh22_Tau38_phi-4_paper.{pdf,png}`

---

### Figure 2 — PSD β Recovery Method B (NH=21, TAU=2, PHI=-3)
```bash
cd plots/psd_beta_recovery_tau2/
python nh_phi_psd_beta_recovery_methodB_paper.py
```
**Requires:** Stage 1 NPZ (Nh21 Tau2), Stage 3 cache files (auto-generated on first run)
**Outputs:** `psd_beta_recovery_Nh21_Tau2_Phi-3_methodB_paper.{pdf,png}`

## Notes

- Always run Stage 1 before Stage 3 or 4 — cache scripts need the NPZ files.
- Stage 3 caches are created automatically; delete them to force regeneration.
- All figures are also available as standalone reproducibility packages under `plots/`.
- This file is updated at the end of each working session.
